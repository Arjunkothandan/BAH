🌌 AI Exoplanet Detection System

**An AI-assisted astronomical data analysis platform designed to detect potential exoplanets from NASA TESS light curve observations.**

**Features:**
🔭 NASA TESS Light Curve Retrieval
📈 Time-Series Signal Processing
🧹 Quality Filtering & Data Cleaning
⚡ Flux Normalization
🌍 Exoplanet Transit Detection
📊 Box Least Squares (BLS) Period Analysis
📉 Transit Depth & Duration Estimation
🔍 Signal-to-Noise Ratio (SNR) Analysis
🎯 Confidence Scoring System
📋 Transit Candidate Evaluation
📊 Interactive Streamlit Dashboard
📈 Light Curve Visualization
🚀 Extensible AI/ML Architecture
**Tech Stack:**
Python
Lightkurve
Astropy
NumPy
Pandas
Matplotlib
Streamlit
Scikit-learn (Upcoming)
Objective

To automate the identification and analysis of exoplanet transit signals from astronomical time-series data and provide researchers with an accessible tool for candidate screening and visualization.
